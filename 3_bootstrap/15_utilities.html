타이포그래픽 / 텍스트 클래스 

            .display-* : 제목에 글씨를 좀더크게 4단계(display-1 ~ 4)
            .font-weight-bold : bold text
            .font-weight-bolder : bolder bold text
            .font-weight-nomal : normal text
            .font-weight-light : light weight text
            .font-weight-lighter : lighter weight text
            .font-italic : italic text
            .lead : 단락을 돋보이게 해줌
            .small : 상위태그에 비해 85%로 설정
            .text-break : 긴 텍스트가 레이아웃을 깨드리는걸 방지
            .text-center : 텍스트를 가운데 정렬
            .text-decoration-none : 언더라인 방지
            .text-left: 텍스트를 왼쪽 정렬
            .text-justify : 왼쪽줄 시작점 오른쪽 끝점을 동일선상에 맞춤
            .text-monospace : 고정폭 텍스트
            .text-nowrap : 줄바꿈 텍스트가 없음을 나타냅니다.
            .text-lowercase : 소문자
            .text-uppercase : 대문자
            .text-capitalize : 워딩에서 시작하는 단어가 대문자로
            .list-unstyled : ul ol에서 숫자나 아이콘 지움
            .list-inline : 가로로 정렬
            .pre-scrollable : pre태그에 스크롤바 생성
