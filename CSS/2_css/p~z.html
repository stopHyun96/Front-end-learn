position

디스플레이 영역에서 엘리먼트의 위치에 대해서 지정

        relative : 상대적 (페런츠 요소에서 영향을 받아 순차적으로 정렬)
        absolute : 절대적 (상위 요소에 영향을 받지 않는다)
        fixed : 고정 (스크롤바를 올려줘도 그자리 그대로 잇음)
        sticky : 스티커처럼 그 주변을 지나갈 때 포지션 변동으로 생성

scroll-behavior

스크롤 되는 박스 모델 직선점프 대신에 스크롤 위치를 부드럽게 애니메이션 할지에 대해서 정해줌

        - auto : 기본값 직선점프
        - smooth : 부드러운 애니메이션 효과
        - inherit : 속성을 초기 기본값으로 설정
