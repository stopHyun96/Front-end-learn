height

요소의 높이를 설정합니다

- pc버전을 만들때는 px
- 모바일과 태블릿을 만들때는 %를 사용하는게 좋습니다
- min-height : 요소의 최소 높이
- max-height : 요소의 최대 높이를 지정합니다
- auto : 요소가 콘텐츠에 올바르게 표시될수 있도록 자동으로 
  조정합니다

image-rendering

이미지 크기 조정에 사용할 알고리즘 유형을 지정합니다

- auto : 브라우저가 스케일링 알고리즘을 선택하도록 해주는것 기본값
- smooth : 이미지의 색상을 부드럽게 해주는 알고리즘
- high-quality : 부드러움은 동일하지만 고품질 스케일링을 선호합니다
- crisp-edges : 이미지의 대비와 가장자리를 유지하는 알고리즘을 사용합니다
- pixelated : 이미지를 확대했을때는 화면에서 거칠게 보이고
이미지를 축소하거나 원래크기일때는(100%)일때는 부드럽게
보이는것

- 사용예시 -
img{
width:100%;
}
.image{
width:100px;
height:100px;
image-rendering:smooth;   
}

2em = 16px x 2 = 32px;

- indent : 들여쓰기
- text-transform:uppercase(대문자),lowercase(소문자)

object-fit

컨테이너에서 지정한 css맞게 img 또는 비디오 크기를 조정하는 방법

- fill : 기본값 크기가 조정죔
- contain : 콘텐츠 상자에 맞게 가로세로 비율을 유지하도록 크기가 조정됩니다
- cover : 지정한 크기만큼 잘려서 보이게 됩니다
- scale-down : 크기가 좀더 작아집니다

overflow:

콘텐츠 내용이 넘칠때 숨기거나 아니면 스크롤바를 생성

- overflow:hidden : 넘치는 부분은 숨김
- overflow:auto : 넘치는 부분은 스크롤바 생성
- overflow-x,overflow-y:가로세로부분을 따로 지정
- overflow-wrap:normal(넘치는 상태로 지속)
  break-word:(넘치는 부분을 끊고 다시시작)
