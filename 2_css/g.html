1) display:grid
    - 상위컨테이너에 display:grid선언
    - grid: 높이 / 칸에 갯수
    - grid-gap : 그리드에 간격 설정

    2) 그리드 정의 :
    - 좀더 복잡한 레이아웃에서 사용하시면 좋습니다
    - 그리드는 flex와 같이 컨테이너와 아이템 두가지 
      개념으로 구분되어 있습니다
    
    3) 이름을 정하고 그걸 활용합니다
    grid-area : 명칭을 정하고 영역을 그만큼 확보합니다
    - 명칭을 정하지 않고 사용할수도 있다
    - 그리드에이리어는 아래 속성에 약어 입니다
    grid-row-start,grid-column-start,grid-row-end,
    grid-column-end

    4) gap : 행과 열사이의 간격 크기를 정의 합니다
    - 아래속성에 약어 입니다
    row-gap,column-gap

grid-auto-flow : 배치항목이 그리드에 삽입되는 방식을 제어합니다 

속성값

- row : 기본값 각 행을 채워 항목배치
- column : 각 열을 채워 항목배치
- dense : 그리드의 구멍을 채우기 위해 항목배치
- row dense : 각 행을 채워 항목배치하고 그리드의 구멍을 채웁니다
- column dense : 각 열을 채워 항목배치하고 그리드의 구멍을 채웁니다
