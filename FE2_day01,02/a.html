animation

            animation-delay : 몇초후에 시작할건지에 대한 지정
            animation-direction : 애니메이션이 어떤 방향으로 시작할지를 정함
            - normal : 기본값 앞으로(오른쪽으로)
            - reverse : 뒤에서부터(오른쪽에서 왼쪽으로)
            - alternate : 왼쪽에서 오른쪽으로 가다가 다시 오른쪽에서 왼쪽으로
            - alternate-reverse : 오른쪽에서 왼쪽으로 가다가 역방향

            animation-duration : 애니메이션이 지속되는 시간을 지정
            animation-fill-mode : 애니메이션이 재생되지 않을 때(시작 전, 종료 후, 또는 둘다)요소의 스타일을 지정
            animation-iteration-count : 애니메이션 반복횟수를 지정
            animation-play-state : 실행시키거나 종료시키거나
            animation-timing-function : 속도관련 빠르거나 느리거나
