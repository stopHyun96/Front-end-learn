생성자가 아니여서 생성하지 않고 객체를 사용해서 호출할 수 있다.

        30)pow() : x의 값을 y의 거듭제곱(Xy)으로 리턴
        

        

        31)random() : 0에서 최대 1(제외)까지 임의의 숫자를 리턴
        

        

        32)round() : 소수점을 기준으로 .5이하이면 내리고 이상이면 올림
        

        

        33)sign() : 숫자가 양수이면 1을 리턴 음수이면 -1 숫자가 0이면 0을 리턴
        

        

        34)sin() : 숫자의 사인을 리턴
        -1과 1사이의 값을 리턴
        

        
        
        35)sinh() : 숫자의 하이퍼볼릭 사인을 리턴
        36)sqrt() : 숫자의 제곱근을 리턴
        37)SQRT1_2 : 1/2의 제곱근을 반환 약 0.707리턴
        38)SQRT2 : 2의 제곱근을 리턴 약 1.414를 리턴
        39)tan() : 숫자의 탄젠트를 리턴
        40)tanh() : 숫자의 하이퍼볼릭 탄젠트를 리턴
        41)trung() : 숫자의 정수부분을 리턴(소수점 이하 자릿수를 제거 숫자를 반올림하지 않음)
