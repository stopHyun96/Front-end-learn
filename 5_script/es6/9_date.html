date

- date객체는 날짜 및 시간과 함께 작동됩니다
- new Date(), new Date(milliseconds), new Date(dateString),
  new Date(year,month,day,hours,minutes,seconds,milliseconds);




- 자바스크립트는 1970년 1월 1일부터 날짜를 밀리초단위로 저장합니다
1970년 1월1일부터 100000000000밀리초는 대략 1973년 3월3일 입니다




- 날짜 강제로 넣기(문자열)




- 날짜 강제로 디테일하게 넣기(넘버)




날짜메서드
- 1) constructor : 날짜 생성자를 가져옴 function Date() { [native code] }
- 2) getDate() : 해당월의 날짜를 가져옴 (날짜의 일자를 1~31을 리턴)





- 3) getDay() : 날짜의 요일을 리턴(인덱스번호로)
     0 ~ 6(일요일)



- 4) getFullYear() : 날짜의 전체년도 (4자리)를 리턴




- 5) getHours() : 날짜의 시간을 리턴(0~23)




- 6) getMilliseconds() : 특정날짜의 밀리초(0~999)를 리턴




- 7) getMinutes() : 날짜의 분(0~59)를 리턴합니다



- 8) getMonth() : 날짜의 월(0~11)을 리턴합니다
1월 = 0, 2월 = 1'''''



- 9) getSeconds() : 날짜의 초를 리턴합니다(0~59)




- 10) getTime() : 1970년 1월1일 00:00:00이후의 밀리초수를 리턴



- getTimezoneOffset():UTC(세계협정시)시간과 현지시간 차이를 리턴합니다
  UTC+9
- GMT(그리니치 표준시)와 동일



- 11) getUTCDate() : 날짜 객체의 일(1~31)을 리턴
- 12) getUTCDay() : 날짜의 요일(0~6)을 리턴
- 13) getUTCFullYear() : 날짜의 연도를 리턴
- 14) getUTCHours(): 날짜의 시간(0~23)을 리턴
- 15) getUTCMiliseconds(): 날짜의 밀리초(0~999)
- 16) getUTCMinutes(): 날짜의 분(0~59)을 리턴
- 17) getUTCMonth() : 날짜의 월(0~11)을 리턴
- 18) getUTCSeconds() : 날짜의 초(0~59)를 리턴
- 19) getYear() : getFullyear로 대체
- 20) now() : 1970년 1월 1일 이후의 밀리초 수를 리턴합니다
구문은 = Date.now();




- 21) parse() : 1970년 1월1일 이후의 시차를 리턴
