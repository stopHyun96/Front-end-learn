웹브라우저에 의해 실행되는 명령입니다

- statement Identifiers (문 식별자)
1)break : 스위치또는 loop에서 제동을 겁니다
- 스위치 또는 루프에서 빠져나갑니다
- 루프내에서 루프를 중단하고 루프가 있는 경우 다음에 코드를 계속 실행합니다



while을 사용할경우




2)class : 자바스크립트 클래스를 시작합니다




3)const : 상수 값으로 변수 선언
-  변수정보를 저장하는 컨테이너



상수객체




4)continue : 지정된 조건이 발생하면 (루프에서)하나의 반복을 중단하고 루프에서 다음반복을 계속합니다
예를 들어 4를 지정하면 4를 건너뛰고 다음조건까지만 나옴
continue(점프오버)와 break(점프아웃)문에 차이점





5)debugger : 자바스크립트 실행을 중지하고 디버깅기능을 호출합니다 f12를 눌러서 확인할수 있음
(사용 가능한 경우에)



6)do...while : 스테이트블록을 실행하고 참인 동안에 블록을 반복



7)for : 코드블록을 여러번 반복합니다



8)for...in : 객체의 속성을 통해서 루프



9)for...of : 반복가능한 객체의 값을 반복합니다




10)function :  함수선언






11)if...else...elseif : 조건에 따라 실행할 블록을 표시
지정된 조건이 참이면 실행되는 코드블럭 
조건이 거짓이면 다른 코드블럭을 실행할수도 있습니다
1) 시간이 20시 미만이면 굿데이를 출력


12)let : 변수선언
13)return : 함수 실행을 중지하고 해당함수에서 값을 반환합니다
14)switch : 다른 경우에 따라 실행될 블록을 표시
15)throw : 오류 발생
16)try catch finally : try블록에서 오류 발생시 실행할 블록을 표시하고 오류처리를 구현합니다
17)var : 변수선언
18)while : 조건이 참인 동안 실행할 명령문 블록을 표시합니다
