math

생성자가 아니여서 생성하지 않고 객체를 사용해서 호출할 수 있다.

        - Math 개체 메서드 및 속성
        1)Math.abs() : 숫자의 절대값을 리턴
        - 숫자가 아닌 경우 NaN
        - 값이 null이면 0리턴

        

        

        2)acos() : 숫자의 아크코싸인(라디안)
        - 0과 PI사이의 값을 리턴
        - -1에서 1사이의 매개변수를 리턴
        - Math.acos(-1) : PI를 리턴
        - Math.acos(0) : PI/2를 리턴
        - Math.acos(1) : 0를 리턴
        - -1 ~ 1 사이를 벗어나거나 숫자가 아닌 경우 NaN을 리턴

        

        

        3)acosh() : 숫자의 쌍곡선 역코사인을 리턴
        - 매개변수가 1보다 작으면 메서드는 NaN을 리턴

        

        

        4)asin() : 숫자의 아크사인(라디안을 리턴)
        - -PI/2와 PI/2 사이의 값을 반환
        - -1에서 1 사이의 매개변수를 리턴
        - Math.asin(-1) -PI/2를 반환.
        - Math.asin(0) 0을 반환.
        - Math.asin(1) PI/2을 반환.

        

        

        

        5)asinh() : 쌍곡선 아크사인에 숫자를 리턴
        - 0일때 0나오고
        

        

        6)atan() : 숫자의 아크탄젠트를 -PI/2와 PI/2 라디안 사이의 값으로 반환

        

        

        7)atan2() : 인수 몫의 아크탄젠트를 PI와 -PI라디안 사이의 숫자값으로 리턴
        반환된 숫자는 양의 x축과 점(x,y)사이의 시계 반대방향 각도(도가 아님)을 나타낸다
        문법 -atn(y,x)
