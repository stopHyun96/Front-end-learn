Global Reference

javascript 전역속성 및 메서드는 javascript 객체와 함께 사용할 수 있다

        1) decodeURI() : URI를 디코딩함
        - http://domain.com?my test/.asp?name=saab
        

        

        2) decodeURIComponent() : 인코딩 후 URI를 디코딩
        3) encodeURI() : 예전에 escape사용
        4) encodeURIComponent() : URI구성요소를 인코딩
        5) eval() : 인수를 평가하거나 실행
        

        

        6) Infinity : 양과 음의 무한대의 숫자
        7) isFinite() : 숫자가 유한한지에 대해서 유한하면 true
        

        

        8) isNan : Not a Number => 숫자가 아니면 true, 숫자면 false
        9) Number() : 값을 숫자로 리턴
        

        

        10) parseFloat() : 값을 문자열로 구문 분석하고 첫번째 숫자를 리턴
        - 첫번째 문자를 반환 할 수 없으면 NaN
        - 선행 및 후행 공백은 무시
        - 발견된 첫번째 숫자만 리턴

        

        

        11)parseInt() : 문자열로 구문분석하고 첫번째 정수를 리턴
        - 기수 매개변수를 사용할 수 체계를 지정
        - 2 = 2진수 8 = 8진수
        - 0x로 시작하면 기수를 16으로 가정

        

        

        12)toString() : 메서드값을 문자열로 반환
        

        
        13) undefined() : 정의되지 않음
