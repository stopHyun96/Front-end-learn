1) 산술연산자 : 변수 및/또는 값 사이의 산술을 수행하는데 사용
        + : x = y + 2
        - : x = y - 2
        * : x = y * 2
        ** : x = y ** 2 (y의 2제곱)
        

        
        / (몫)
        % (나머지)
        

        

        ++(왼쪽) : 증가
        ++(오른쪽) : 증가
        --(왼쪽) : 감소
        --(오른쪽) : 감소

        2) 할당연산자 : 변수에 값을 할당하는데 사용
        =
        +=
        

        
        -=
        *=
        /=
        %=
        :
        

        

        3) 문자열 연산자
        문자열을 더해주는 것
        + +=

        4) 비교 연산자
        == : 같다
        === : 타입까지 고려해서 같다
        != : 같지 않다
        !== : 값 또는 유형이 같지 않다
        > : 크다
        < : 작다
        >= : 크거나 같다
        <= : 작거나 같다
        
        

        

        5) 조건부(삼항) 연산자 : 조건에 따라 변수에 값을 할당
        
        
        

        

        6) 논리연산자 : 변수 또는 값 사이의 논리를 결정하는데 사용
        && = 그리고 (x < 10 && y > 1)
        

        
        || = or
        

        
        ! = 아님 같지 않으면 true 같으면 false
        

        

        7)비트 연산자
        & AND x = 5 & 1 => 0101 & 0001 = 0001
        | OR x = 5 | 1 => 0101 | 0001 = 0101
        ~ NOT x = ~5 = 1010
        ^ XOR x = 5 ^ 1 = 0100
        << left shift x = 5 << 1 1010
        >> Right shift x = 5 >> 1 0010

        8) 삭제 연산자 : delete
        9) in 연산자 : 객체가 있으면 true
        10) instanceof : 지정된 객체인 경우 true 를 리턴
        11) void : 표현식을 평가하고 undifinde를 리턴
        이연산자는 종종 void(0) : 반환값을 사용하지 않고 식을 평가하는데 유용
        정의되지 않은 기본값을 얻는데 사용
