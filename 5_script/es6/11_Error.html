Error Object

        error 객체는 오류가 발생할 때 오류 정보를 제공한다.
        - Error 명
        EvalError : 평가 오류 (더 이상 사용하지 않음)
        RangeError : "범위를 벗어난 숫자 발생했습니다"
        ReferenceError : 참조 오류 ("잘못된 참조가 발생했습니다")
        SyntaxError : 구문 오류가 발생했습니다
        TypeError : 유형 오류
        URIError : encodeURI에서 오류가 발생했습니다

        - Error Object Properties
        name : 에러명
        message : 오류메세지
