8)find() : 테스트를 통과한 첫번째(먼저순서의 인덱스) 요소의 값을 리턴


9) findIndex() : 테스트를 통과한 첫번째 요소의 인덱스(위치)를 리턴





10)forEach() : 인덱스번호가 나오면서 리스트로 출력이됨
0: apple
1: orange
2: cherry

11)from() : 길이속성이 있는  객체에서 배열을 리턴
- 공백까지 인식




12)indexOf() : 검색한 해당키워드에 첫번째 인덱스를 리턴




13)isArray() : 객체가 배열이면 T, 아니면 F
Array.isArray(obj)

14)join() : 기존애 배열을 변경하지 않고 워딩사이에 무언가 추가할때 사용
["Banana", "Orange", "Apple", "Mango"]; => Banana,Orange,Apple,Mango






15)keys() : 배열을 인덱스키(숫자)로 리턴 키가 있는 Array Iterator 객체
