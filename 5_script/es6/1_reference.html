참조 카테고리

- array : 자바스크립트 배열객체 
- boolean : true 또는 false
- classes : class정의하여 좀더 편하게 사용
- date : 날짜및 시간
- error : 오류객체
- global : 전역매서드 및 속성
- json : JavaScript Object Notation (자바스크립트 객체 표기법)
- math : 수학객체
- number : 숫자
- object : 객체 자바스크립트에 데이터 유형중에 하나
- operators : 연산자(+,-)
- precedence : 연산자 우선 순위
- regexp : 정규 표현식
- statements : 문(실행하라는 지시)
- string : 문자열
- typedArray : 형식배열
