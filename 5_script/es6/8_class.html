class

- ES6지원
- 익스플로러라는 브라우저에서 예전에는 사용불가능
- class를 사용하는 가장큰이유는 : 재사용성
작은사이트를 만들때는 function하나로 개발이 용이했다면
좀더 복잡한 사이트를 만들때 class가 필요함
- 함수유형이지만 키워드를 사용해서 속성은 constructor메소드 내부에 할당됩니다

ex)car클래스를 만든 다음 car클래스를 기반으로 mycar라는 객체를 만듭니다





- constructor() : 클래스 네에서 생성된 객체를 만들고 초기화하기 위한 특수메서드입니다
class는 하나이상의 constructor()매서드를 가질수 없습니다 syntaxError가\
됩니다

- extends :  다른클래스(부모)의 자식클래스를 만드는데 사용됩니다




- static : 클래스에 대한 정적매서드를 정의합니다
클래스의 인스턴스/객체를 생성하지 않고 클래스에서 직접호출됩니다






- super : 부모의 클래스생성자를 호출하여 부모의 속성및 메서드에 엑세스 하는데 
사용됩니다.
