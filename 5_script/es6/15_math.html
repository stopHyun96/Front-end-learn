생성자가 아니여서 생성하지 않고 객체를 사용해서 호출할 수 있다.


        8)atanh() : 숫자의 하이퍼볼릭 아크탄젠트를 리턴
            - -1과 1사이의 숫자
    
        

        

        9)cbrt() : 숫자의 세 제곱근을 리턴

        

        

        10)ceil() : 숫자를 반올림하여 가장 가까운 정수로 반올림한다

        

        

        11)clz32() : 32비트 이진수에서 선행 0의 수를 리턴
        - countleadingzeroes32의 약자 숫자가 0이면 32를 리턴

        

        

        12)cos() : 숫자의 코싸인을 리턴, -1과 1사이의 숫자를 리턴, 라디안 단위의 숫자를 리턴
        0 => 0 => 0
        1 => PI/180 => 0.0175
        360 => PI*2 => 6.28

        

        

        13)cosh() : 숫자의 하이퍼볼릭 코싸인을 리턴
        
        

        

        14)E : 오일러수를 리턴
        - 약 2,718을 리턴
        

        
        
        15)exp() : E x 의 값을 반환
        여기서 E는 오일러 수 (약 2.7173)이고 x는 전달된 수이다
        
        16)expm1() : E x 빼기 1의 값을 반환
        여기서 E는 오일러 수 (약 2.7173)이고 x는 전달된 수이다
        
        17)floor() : 숫자를 가장 가까운 정수로 내림한다

        

        

        18)fround() : 숫자의 가장 가까운 32비트 단정밀도 부도 소수점을 리턴

        

        

        19)LN2 : 자연 로그를 리턴 약 0.693을 리턴
        20)LN10 : 10의 자연 로그를 리턴 약 2.302 리턴
        21)log() : 자연로그(밑이 E인 것)를 리턴
        22)log10() : 숫자의 밑이 10인 로그를 리턴
        23)log1p() : 1+ 숫자의 자연로그(밑E)를 리턴
        24)log2() : 숫자의 밑이 2인 로그를 리턴
        25)LOG2E : E의 밑이 2인 로그를 리턴
        - 약 1.142를 리턴
        26)LOG10E : E의 밑이 10인 로그를 리턴
        27)max() : 가장 높은 값을 가진 숫자를 리턴
        28)min() : 가장 낮은 값을 가진 숫자를 리턴

        

        

        29)PI : 3.14를 리턴
