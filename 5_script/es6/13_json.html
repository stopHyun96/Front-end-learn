JSON(Java Script Object Notation)

        - 데이터를 저장하고 전송하기 위한 형식
        - 텍스트고 어디든 전송될 수 있고 모든 프로그래밍 언어로 읽을 수 있다
        - 자바스크립트 객체는 json으로 변환할 수 있고 json은 다시 자바스크립트로 변환 할 수 있다
        - 이렇게 하게되면 구문분석이나 번역없이 javascript객체로 사용할 수 있다

        json메서드
        - parse() : 문자열을 구문분석하고 자바스크립트 객체를 리턴
        - stringify() : 자바스크립트 객체를 json문자열로 리턴
