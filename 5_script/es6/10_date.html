- setMiliseconds() : 날짜의 밀리초를 설정
        - setMinutes() : 날짜의 분을 설정
        - setMonth() : 날짜의 월을 설정(0부터 시작)
        - setSeconds(): 날짜의 초를 설정
        - setTime() : 1970년 1월 1일 자정부터 지정된 밀리초수를 더하거나 빼서 날짜와 시간을 설정
        - setUTCDate() : 메서드는 UTC에 따라 Date 객체의 날짜를 설정
        - setUTCFullYear() : UTC에 따라 날짜 개체의 연도를 설정
        - getUTCHours(): 날짜의 시간(0~23)을 리턴
        - getUTCMiliseconds(): 날짜의 밀리초(0~999)
        - getUTCMinutes(): 날짜의 분(0~59)을 리턴
        - getUTCMonth() : 날짜의 월(0~11)을 리턴
        - getUTCSeconds() : 날짜의 초(0~59)를 리턴
        - getYear() : getFullyear로 대체
        - todateString() : 메서드는 날짜 객체의 날짜
        

        
        
        - toGMTString() : 사용안함 => toUTCString() : UTC에 따라 날짜 개체를 가져옴
        - toISOString() : ISO표준을 사용하여 날짜 객체를 문자열로 리턴 (YYYY-MM-DDTHH:mm:ss.sssZ)
        

        
        - toJSON() : 메서드는 날짜 개체를 JSON날짜 형식의 문자열로 반환
        JSON 날짜는 ISO-8601 표준과 동일한 형식(YYYY-MM-DDTHH:mm:ss.sssz)를 가진다.

        - toLocaleDateString() : 로케일 규칙을 사용하여 날짜 객체의 날짜를 문자열로 리턴
        - toLocaleTimeString() : 로케일 규칙을 사용하여 날짜 객체의 시간 부분을 문자열로 리턴
        

        

        - toLocaleString() : 메서드는 로케일 설정을 사용하여 Date 객체를 문자열로 반환한다.
        - toString() : 날짜 개체를 문자열로 반환
        - toTimeString() : 날짜 객체의 시간 부분을 문자열로 반환
