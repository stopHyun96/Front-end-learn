객체지향언어 : 다수의 객체로 만들고 이들끼리 서로 상호작용하도록 만든 프로그래밍 언어

- 원시값(pri,itive,또는 원시자료형)을 제외한 모든것이 객체
- 원시값에 종류(객체가 아니면서 매서드도 가지지 않는 타입)
=>string,number,bigint,boolean,undefined,symbol,null

객체지향언어 특징
- 캡슐화 : 데이터와 알고리즘을 하나로 묶어서 하나의 캡슐처럼 만듬
- 상속 : 부모가 자식에게 유전자를 물려주듯 상위클래스의 
         모든것을 하위 클래스가 이어받음
- 다형성 : 하나의 객체가 다른 여러객체로 재구성
- 추상화 : 프로그램을 만드는데 필요한 부분만 파악해 추출하고    필요하지 않는것은 제거

객체지향언어 장점
- 재사용성 : 상속이 가능하기 때문에 코드를 재사용함
- 생산성증가 : 독립된 객체로 이루어져 있기때문에 생산적으로 작업할수있고 유지 및 보수가 용이 합니다
- 자연스러운 모델링 : 객체는 세상에 존재하느,모든것 입니다
