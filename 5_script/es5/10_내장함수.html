문법 : eval(string)

isFinite() : 매개변수가 유한한 수인지 검사

문법 : isFinite(value)
- 유한한값이 숫자이면 true이고 아니면 false

isNaN() : 매개변수가 숫자가 아닌지를 검사하는 함수

- nan (not a number)
- 문법 : isNaN(value)
- 숫자가 아니면 true이고 숫자이면 false

Number() : 문자열을 숫자로 변환하는 함수

숫자로 변환할수없는 값인경우 nan을 리턴

parseFloat(),parseInt() : 문자열을 수로 바꾸는 함수

parseFloat() : 문자열을 실수로 바꾸는 함수

수로시작할때 그수를 실수로 바꾸고 띄어쓰기가 있을때는 첫번째수만 바꿈 공백으로 시작하면 공백은 무시 수가 아닌문자로 시작하면 NaN을 리턴

parseInt() : 문자열을 정수로 바꾸는 함수

- 문법 : parseInt(string, n)
- string을 n진법일때 값으로 바꿉니다 n옵션으로 2부터 36까지 입력할수 있습니다.입력하지않으면 10으로 처리
- 소수부분은 버림
- 0x로 시작하면 16진법으로 처리

parseint('100', 2) : 100은 2진법으로 4입니다.
parseint('0x100') : 100은 16진법으로 256입니다.

String : 숫자를 문자열로 변환하는 함수
