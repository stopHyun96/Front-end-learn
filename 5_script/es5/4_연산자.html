연산자:비교연산자

변수의 값을 어떤 값 또는 다른 변수와 비교할 때 사용하는 것이 비교 연산자(Cpmparison Operator)입니다. 
비교 연산자는 8가지가 있는데, 각 연산자의 의미를 알아보겠습니다.  

1) == : 같다 변수타입과 상관없이 같으면 참이다
5 == 5
'5' == 5

2)!= '같지 않다'
6 != 5

3) === : 엄격하게 같다, '엄격하게'의 의미는 변수 타입까지 고려한다는 것입니다. 
5 === 5
'5' === 5

4)!== : 엄격하게 같지 않다 변수타입까지 같지 않아야 됨
'5' !== 5 (true)

5) > = 크다
6) >= 크거나 같다 둘중에 하나만 맞으면 성립
7) < = 작다
8) <= 작거나 같다 둘중에 하나만 맞으면 성립

연산자:type of 연산자

- 변수의 데이터 타입을 반환하는 연산자입니다.
반환되는 값은 다음과 같다
- undefined : 변수가 정의되지 않거나 값이 없을 때
- number : 데이터 타입이 수일 때
- string : 데이터 타입이 문자열일 때
- boolean : 데이터 타입이 불리언일 때
- object : 데이터 타입이 함수, 배열 등 객체일 때
- function : 변수의 값이 함수일 때
- symbol : 데이터 타입이 심볼일 때
